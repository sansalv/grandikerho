# Grandikerhon rankaisija

## Description
Tämä koodi kertoo myöhästyneelle sanktion suuruuden.

$f(t) = \log_5(1+t)+0.05t$ <br>
$R(t) = 0.5(\sin(15t)^2+1)$<br>
$g(t) = f(t)R(t)$ <br>
